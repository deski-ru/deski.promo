Установка чата для сайта Deski.ru на WordPress

Как установить код чата для сайта Deski.ru на сайт WordPress

Установка через виджеты

Откройте панель редактирования сайта
Выберите пункт меню: Внешний вид -> Виджеты
Кликните "HTML-код", выберите "Подвал 1" (или "Footer 1") и нажмите "Добавить виджет"

Вставьте код чата для сайта в поле "Содержимое" и нажмите "Сохранить"

Перейдите на сайт и проверьте правильность отображения онлайн консультанта
