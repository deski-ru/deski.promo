Приложения

Вы можете установить приложение Deski.ru себе на компьютер с Windows 7, 8, 10 или на телефон с Android.
Наше приложение для Windows поддерживат весь функционал личного кабинета Deski.ru и позволяет получать мгновенные уведомления о новых соощениях.
В приложение для Android реализован функционал для обмена сообщениями с клиентами и получение мгновенных уведомлений о новых сообщениях.

Windows 10

Deski.ru.10.x64.zip (рекомендуется)
Deski.ru.10.x86.zip

Windows 7, 8

Deski.ru.7.x64.zip (рекомендуется)
Deski.ru.7.x86.zip
