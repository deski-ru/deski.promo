Приложение для Windows

Вы можете установить приложение Deski.ru себе на компьютер
Наше приложение поддерживат весь функционал личного кабинета Deski.ru
Так же, позволяет получать уведомления о новых сообщениях

Установка

Установка приложений происходит через браузеры Google Chrome и Microsoft Edge
Для установки необходимо в любом из указанных браузеров перейти в личный кабинет https://lk.deski.ru, открыть меню браузера и нажать "Установить Deski.ru - система HelpDesk" (см. скрин ниже)
После установки приложения появятся ярлыки в меню Пуск и на рабочем столе

Включение уведомлений

Откройте установленное приложение. Нажмите в левом или верхнем меню "Уведомления" и подтвердите действие во всплывшем сообщении (см. скрин ниже)
